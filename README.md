4) Git
    - git commit -m "Описание изменений"
    - git add  Добавление всех изменённых файлов в индекс
    - git status Просмотр состояния репозитория и изменений
    - git log Просмотр истории коммитов
    - git push origin <ветка> Отправка коммитов на удалённый репозиторий
    - git fetch Получение изменений с удалённого репозитория без слияния
    - git pull Получение изменений с удалённого репозитория с слиянием
    - git branch   Просмотр всех локальных веток
    - git checkout <ветка> Переход на ветку
    - git merge <ветка> Объединить ветки
    - git checkout <ветка> Переключение на ветку
    - git rebase <ветка> Перенос коммитов текущей ветки на базу другой ветки
    - git rm <файл> Удаление файла и добавление удаления в индекс
    - git cherry-pick <commit-hash> Взять отдельный коммит из одной ветки и применить его в другой
